Chatbox Entegrasyonu

Bu sayfada chatbox widget'ı entegre edilmiştir. Sağ alt köşedeki chatbox butonuna tıklayarak canlı destek alabilirsiniz.

Entegrasyon Kodu:

<!-- Chatbox widget'ını sayfanıza ekleyin --> <script src="chatbox.js"></script> <script> // Chatbox'ı özelleştirerek başlatın const chatbox = new ChatboxWidget({ welcomeMessage: 'Merhaba! Size nasıl yardımcı olabilirim?', quickReplies: [ 'Ürünler hakkında', 'Sipariş durumu', 'İade politikası', 'Kargo bilgileri' ], responses: { 'Ürünler hakkında': 'Ürünlerimiz hakkında detaylı bilgi için kategoriler bölümünü inceleyebilirsiniz.', 'Sipariş durumu': 'Sipariş durumunuzu öğrenmek için hesabınıza giriş yapabilirsiniz.', // Daha fazla özel cevap ekleyebilirsiniz } }); // Chatbox'ı programatik olarak kontrol edebilirsiniz // chatbox.open(); // Chatbox'ı aç // chatbox.close(); // Chatbox'ı kapat // chatbox.setResponses(newResponses); // Cevapları güncelle </script>

Özellikler:

  • ✅ Sağ alt köşede sabit konum
  • ✅ Hazır cevaplar (Quick Replies)
  • ✅ Kullanıcı mesaj girişi
  • ✅ Responsive tasarım
  • ✅ Programatik kontrol
  • ✅ Özelleştirilebilir tema
  • ✅ E-ticaret odaklı cevaplar
